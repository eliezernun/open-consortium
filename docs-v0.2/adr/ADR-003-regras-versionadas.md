# ADR-003 — Regras versionadas

- Status: Aceito
- Data: 2026-07-13

## Decisão

Toda regra regulatória ou contratual executável possui identificador, versão semântica e vigência.

## Consequências

- assembleias antigas podem ser reproduzidas;
- alterações legais não sobrescrevem regras anteriores;
- implantação deve suportar coexistência de versões;
- catálogo regulatório torna-se artefato de release.
