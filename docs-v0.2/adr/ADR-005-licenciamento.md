# ADR-005 — Licenciamento

- Status: Proposto
- Data: 2026-07-14

## Decisão

Adotar **Apache License 2.0** para o código e, preferencialmente, **CC BY 4.0** para a documentação.

## Consequências

A escolha facilita adoção empresarial e integração proprietária, mas permite forks privados e ofertas SaaS fechadas.

## Mitigações

- política de marcas;
- suíte oficial de conformidade;
- governança pública;
- programa de compatibilidade;
- DCO;
- releases oficiais identificados.
